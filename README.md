# 📘 Clojure Exercises (Exercism)

This repository contains my solutions to [Exercism's Clojure track](https://exercism.org/tracks/clojure).  
It serves as a personal study log and backup of my learning process while practicing functional programming with Clojure.

---

## 🚀 Goals
- Practice functional programming concepts.  
- Learn idiomatic Clojure syntax and style.  
- Improve problem-solving skills with real coding challenges.  
- Build consistency through small daily exercises.

---

Each folder represents an exercise downloaded from Exercism and contains:
- `src/` → the solution code  
- `test/` → the provided unit tests  

--- 

📖 Learning Notes
While solving these exercises, I’m focusing on:

Using higher-order functions like map, filter, reduce.

Understanding threading macros (->, ->>).

Working with immutable data structures.

Practicing recursion and laziness.


---

Make sure you have **Clojure CLI tools** installed:
